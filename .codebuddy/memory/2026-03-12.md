## charging-sim 文件拆分

将 1200 行的单文件 `index.html` 拆分为 4 个文件：

| 文件 | 职责 | 行数(约) |
|------|------|---------|
| `style.css` | 所有 CSS 样式 | ~370 |
| `engine.js` | 车型配置、充电曲线引擎、功率分配器（纯函数，无 Vue 依赖） | ~170 |
| `app.js` | Vue 3 Composition API 应用：状态管理、模拟引擎、UI 交互、Canvas 图表 | ~310 |
| `index.html` | HTML 模板 + 外部文件引用（`<link>` / `<script src>`） | ~240 |

加载顺序：`style.css` → Vue CDN → `engine.js` → `app.js`（engine 必须在 app 之前加载，因为 app 使用 engine 暴露的全局变量）
