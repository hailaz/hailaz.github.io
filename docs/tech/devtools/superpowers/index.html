Superpowers

https://github.com/obra/superpowers

Superpowers жҳҜдёҖдёӘдё“дёә AI зј–зЁӢд»ЈзҗҶпјҲClaude CodeгҖҒCursor зӯүпјүи®ҫи®Ўзҡ„иҪҜд»¶ејҖеҸ‘е·ҘдҪңжөҒжЎҶжһ¶пјҢ69.8k StarгҖӮ

ж ёеҝғжҖқи·ҜпјҡжҠҠ AI д»ЈзҗҶеҪ“дҪңдёҖеҗҚеҲқзә§е·ҘзЁӢеёҲпјҢйҖҡиҝҮдёҖеҘ—еҸҜз»„еҗҲзҡ„"жҠҖиғҪ"е’ҢжөҒзЁӢи§„иҢғпјҢеј•еҜј AI йҒөеҫӘжҲҗзҶҹзҡ„е·ҘзЁӢж–№жі•и®әпјҲTDDгҖҒYAGNIгҖҒDRYпјүиҝӣиЎҢејҖеҸ‘пјҢиҖҢдёҚжҳҜзӣҙжҺҘи®©е®ғеҶҷд»Јз ҒгҖӮ

е·ҘдҪңжөҒзЁӢвҖӢ

Superpowers е°ҶејҖеҸ‘иҝҮзЁӢеҲҶдёәејәеҲ¶жҖ§зҡ„йҳ¶ж®өпјҢжҜҸдёӘйҳ¶ж®өз”ұзү№е®ҡжҠҖиғҪиҮӘеҠЁи§ҰеҸ‘пјҡ

  1. еӨҙи„‘йЈҺжҡҙ вҖ” зј–з ҒеүҚжҝҖжҙ»пјҢйҖҡиҝҮжҸҗй—®з»ҶеҢ–йңҖжұӮпјҢжҺўзҙўжӣҝд»Јж–№жЎҲ
  2. Git Worktree вҖ” и®ҫи®Ўжү№еҮҶеҗҺпјҢеңЁж–°еҲҶж”ҜеҲӣе»әйҡ”зҰ»е·ҘдҪңеҢә
  3. зј–еҶҷи®ЎеҲ’ вҖ” е°Ҷе·ҘдҪңжӢҶи§Јдёәеҫ®е°Ҹд»»еҠЎпјҲжҜҸдёӘ 2-5 еҲҶй’ҹпјүпјҢеҢ…еҗ«ж–Үд»¶и·Ҝеҫ„гҖҒе®Ңж•ҙд»Јз Ғе’ҢйӘҢиҜҒжӯҘйӘӨ
  4. еӯҗд»ЈзҗҶејҖеҸ‘ вҖ” дёәжҜҸдёӘд»»еҠЎеҲҶжҙҫеӯҗд»ЈзҗҶпјҢиҝӣиЎҢдёӨзә§е®ЎжҹҘпјҲи§„иҢғз¬ҰеҗҲжҖ§ + д»Јз ҒиҙЁйҮҸпјү
  5. TDD вҖ” ејәеҲ¶зәў-з»ҝ-йҮҚжһ„еҫӘзҺҜпјҡеҶҷеӨұиҙҘжөӢиҜ• вҶ’ зңӢеӨұиҙҘ вҶ’ еҶҷжңҖе°Ҹд»Јз Ғ вҶ’ зңӢйҖҡиҝҮ вҶ’ жҸҗдәӨ
  6. д»Јз Ғе®ЎжҹҘ вҖ” еҜ№з…§и®ЎеҲ’е®ЎжҹҘпјҢжҢүдёҘйҮҚзЁӢеәҰжҠҘе‘Ҡй—®йўҳпјҢе…ій”®й—®йўҳйҳ»жӯўиҝӣеәҰ
  7. е®ҢжҲҗеҲҶж”Ҝ вҖ” йӘҢиҜҒжөӢиҜ•пјҢжҸҗдҫӣеҗҲе№¶/PR/дҝқз•ҷ/дёўејғйҖүйЎ№пјҢжё…зҗҶ Worktree
